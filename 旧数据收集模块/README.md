## 🆕 新的数据收集方案

请使用项目根目录的 **AI_collect_dual_kline.py**：

```bash
python AI_collect_dual_kline.py
```

**优势**：
- ✅ 使用浏览器拦截，不会被限流
- ✅ 同时收集小时K线和日K线
- ✅ 自动映射为本地ID
- ✅ 支持增量更新

**数据目录**：
- `data/hourly/` - 小时K线
- `data/daily/` - 日K线
